# <a name="dictionaryhomepage-element"></a>DictionaryHomePage 要素

辞書のホーム ページの URL を指定します。

**アドインの種類:** 作業ウィンドウ

## <a name="syntax"></a>構文

```XML
<DictionaryHomePage DefaultValue="URL" />
```
