# <a name="requestedwidth-element"></a>RequestedWidth 要素

コンテンツ アドインの初期幅をピクセル単位 (32 から 1000 まで可能) で指定します。

**アドインの種類:** コンテンツ、メール

## <a name="syntax"></a>構文

```XML
<RequestedWidth>integer</RequestedWidth>
```
