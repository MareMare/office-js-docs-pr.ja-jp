# <a name="equivalentaddins-element"></a>EquivalentAddins 要素

同等の COM アドイン、XLL、または両方との下位互換性を指定します。

**アドインの種類:** 作業ウィンドウ、カスタム関数

## <a name="syntax"></a>構文

```XML
<EquivalentAddins>
...  
</EquivalentAddins>  
```

## <a name="must-contain"></a>含める必要があるもの
