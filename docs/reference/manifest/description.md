# <a name="description-element"></a>Description 要素

Office アドインの説明を 250 文字以下の文字列で指定します。

**アドインの種類:** コンテンツ、作業ウィンドウ、メール

## <a name="syntax"></a>構文

```XML
<Description DefaultValue="string">
```
