# <a name="phonesettings-element"></a>PhoneSettings 要素

このメール アドインが電話で使用されるときに適用するソースの場所と制御の設定を指定します。

**アドインの種類:** メール

## <a name="syntax"></a>構文

```XML
<PhoneSettings>
   ...
</PhoneSettings>
```
