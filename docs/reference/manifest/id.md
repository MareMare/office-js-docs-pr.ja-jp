# <a name="id-element"></a>Id 要素

Office アドインの一意の ID を GUID の形で指定します。

**アドインの種類:** コンテンツ、作業ウィンドウ、メール

## <a name="syntax"></a>構文

```XML
<Id>string</Id>
```
