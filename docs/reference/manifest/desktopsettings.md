# <a name="desktopsettings-element"></a>DesktopSettings 要素

このメール アドインがデスクトップ コンピューターで使用されるときに適用するソースの場所と制御の設定を指定します。

**アドインの種類:** メール

## <a name="syntax"></a>構文

```XML
<DesktopSettings>
   ...
</DesktopSettings>
```
