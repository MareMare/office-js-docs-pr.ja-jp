# <a name="version-element"></a>Version 要素

Office アドインのバージョンを指定します。

**アドインの種類:** コンテンツ、作業ウィンドウ、メール

## <a name="syntax"></a>構文

```XML
<Version>n .n .n .n</Version>
```

## <a name="contained-in"></a>含まれる場所

[OfficeApp](officeapp.md)
