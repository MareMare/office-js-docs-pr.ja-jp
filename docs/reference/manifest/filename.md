# <a name="filename-element"></a>FileName 要素

Web アドインのカスタム関数に対応する XLL のファイル名を指定します。

**アドインの種類:** カスタム関数

## <a name="syntax"></a>構文

```XML
    <FileName>string</FileName>  
```
