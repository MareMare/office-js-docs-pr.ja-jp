# <a name="progid-element"></a>ProgId 要素

Web アドインの作業ウィンドウの同等の COM アドインのプログラム識別子を指定します。

**アドインの種類:** 作業ウィンドウ

## <a name="syntax"></a>構文

```XML
    <ProgId>string</ProgId>  
```
