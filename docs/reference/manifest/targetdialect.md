# <a name="targetdialect-element"></a>TargetDialect 要素

この辞書でサポートされる地域言語を定義します。これはカルチャ名文字列として表現されます。

**アドインの種類:** 作業ウィンドウ

## <a name="syntax"></a>構文

```XML
<TargetDialect>
   string 
</TargetDialect>
```

## <a name="contained-in"></a>含まれる場所

[TargetDialects](targetdialects.md)

## <a name="remarks"></a>注釈

`en-US` などの BCP 47 言語タグの書式で値を指定します。
