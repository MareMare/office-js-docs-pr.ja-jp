# <a name="citationtext-element"></a>CitationText 要素

この辞書の引用定型句を指定します。

**アドインの種類:** コンテンツ、作業ウィンドウ、メール

## <a name="syntax"></a>構文

```XML
<CitationText DefaultValue="string" />
```
