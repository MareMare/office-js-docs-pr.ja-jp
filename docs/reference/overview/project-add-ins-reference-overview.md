# <a name="javascript-api-for-project"></a>JavaScript API for Project 

Projectでは、JavaScript API で作成されたアドインがサポートされていますが、現在、Project との対話専用に設計された JavaScript API はありません。 [共通 API](/javascript/api/office)を使用して Project を作成できます。共通 API の詳細情報については、「[JavaScript API for Office](../javascript-api-for-office.md)」を参照してください。 

## <a name="learn-about-api-capabilities"></a>API 機能について学ぶ

実地体験で Project のコンテンツと対話するための共通 API を使用する場合は、「[Project アドインのクイック スタート](../../quickstarts/project-quickstart.md)」を完了します。
